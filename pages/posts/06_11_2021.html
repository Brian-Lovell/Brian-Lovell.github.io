Coding a Dynamic Landing Page in JavaScript

Author: Brian Lovell
Title: Padawan Web Dev

Linked In Profile Twitter Facebook

This JavaScript project dynamically creates a section menu based on the sections present in the the main index.html home page. The default link behavior is then replaced with JavaScript that scrolls smoothly down to the section. Then when a section is in view for the reader it will change the class of the section to be Active. This will then enable an animated background.

Global Variables

The variable sectionList creates an HTML collection we can acess to retreive information such as the section headers and Id information.

The variable navbarList will allow us to acess the menu.

Functions

The buildNav function will use the sectionList global variable to get the name of the section from the header. Then it will pass the sectiond ID and the header name to the addMenu function.

The addMenu function will create the HTML elements for the navigation menu. It will also add the class, styles, and hyperlink functionality. Event listeners are then added to replace the default hyperlink behavior and smoothly scroll the page down to the section.

The changeActiveClass function adds an event listener for scrolling. When a section is in view, that section gets marked active for unqiue styling and animations from CSS.
